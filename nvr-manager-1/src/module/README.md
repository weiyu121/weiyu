# 子模块设计
子模块中可以参考如下结构：
* view.py: HTTP接口的定义，必须存在
* model[*]: 包含所有sqlalchemy的orm类，一个类对应一张表【注意：实现数据库迁移升级所使用的插件为flask-migrate，它识别不了更改表名、列名这样的操作，他会认为你是删除了原字段后新增一个字段，所以原来的数据自然会丢失，所以不要进行这样的操作，如果必须要进行这样的操作，那就要慎重考虑了】
* exception.py[*]: 包含所有继承自NVRManagerError的异常类，每个异常类对应一种业务异常情况，应该有自己的错误码
* (xxx-)service.py[*]: 子模块主要功能定义
另外，子模块中还可以包含子模块，初始化时会自动搜索所有文件夹里的view.py文件，并统一导入
